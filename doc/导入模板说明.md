# 家族成员批量导入模板说明

## 模板结构

Excel 文件包含三个工作表：

### 工作表 1：成员列表 (Members)

| 字段名 | 必填 | 说明 | 示例 |
|--------|------|------|------|
| 姓名 | 是 | 成员姓名 | 张三 |
| 小名 | 否 | 昵称/小名 | 阿三 |
| 性别 | 是 | 下拉选择：male/female/other | male |
| 生日 | 否 | YYYY-MM-DD 格式 | 1980-05-20 |
| 工作地 | 否 | 工作/居住地 | 北京 |
| 简介 | 否 | 个人简介 | 家族长子 |
| 照片URL | 否 | 照片链接（可选） | http://example.com/photo.jpg |

**注意**：
- 第一行为表头，不要修改
- **性别字段已支持下拉选择**，点击单元格右侧箭头即可选择
- 生日必须是 YYYY-MM-DD 格式

### 工作表 2：关系列表 (Relations)

| 字段名 | 必填 | 说明 | 示例 |
|--------|------|------|------|
| 成员A姓名 | 是 | 关系起始成员姓名 | 张三 |
| 成员B姓名 | 是 | 关系目标成员姓名 | 张小三 |
| 关系类型 | 是 | 下拉选择：parent-child/spouse/sibling | parent-child |
| 具体关系 | 否 | 下拉选择详细关系类型 | father-son |

**关系类型说明**（下拉选项）：
- `parent-child`：父母-子女（血缘关系）
- `spouse`：夫妻（婚姻关系）
- `sibling`：兄弟姐妹

**具体关系可选值**（下拉选项）：
- 血缘关系：`father-son`（父子）、`father-daughter`（父女）、`mother-son`（母子）、`mother-daughter`（母女）
- 婚姻关系：`husband-wife`（夫妻）
- 兄弟姐妹：`brother-brother`（兄弟）、`brother-sister`（兄妹）、`sister-sister`（姐妹）

### 工作表 3：选项说明

包含所有下拉选项的中英文对照表，供参考使用。

## 导入规则

1. **成员去重**：如果姓名已存在，会跳过该成员
2. **关系匹配**：通过姓名匹配成员，如果找不到对应成员会报错
3. **关系校验**：系统会自动校验关系类型和具体关系的匹配性（如选择 spouse 时具体关系只能是 husband-wife）
4. **顺序要求**：建议先导入成员，再导入关系；或者成员和关系同时导入
5. **照片支持**：
   - 可以填写照片的 URL 链接
   - 也可以留空，导入后在系统中单独上传

## 导入流程

1. 下载空白模板（系统提供）
2. 按模板格式填写数据
3. 在系统中上传 Excel 文件
4. 预览导入结果，确认无误后确认导入
5. 导入完成后可在图谱中查看

## 注意事项

- 导入前请备份现有数据
- 建议分批导入，每次不超过 100 条记录
- 确保姓名唯一，避免重名导致关系匹配错误
- 导入过程中请勿关闭页面
